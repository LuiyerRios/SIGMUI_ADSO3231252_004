# SIGMUI – Smart Urban Mobility Management System

SIGMUI is a web-based system designed to manage regulated urban parking zones.  
The platform allows users to register, manage vehicles, reserve parking spaces, and simulate payments through a secure client–server architecture.

---

## 📌 Project Overview

SIGMUI aims to improve urban parking management by providing a digital solution for reservation and control of regulated parking zones.  
The system follows a three-layer architecture (Presentation, Business Logic, and Persistence) to ensure scalability, maintainability, and security.

This project was developed as part of the ADSO program – SENA (2026).

---

## 🏗 System Architecture

The system is based on a client–server model with three logical layers:

- **Presentation Layer (Frontend)** – Web/Mobile interface.
- **Business Logic Layer (Backend)** – REST API, services, authentication, and business rules.
- **Persistence Layer** – Relational database management.

The backend communicates with external services such as:
- Google Maps API
- Simulated Payment Gateway
